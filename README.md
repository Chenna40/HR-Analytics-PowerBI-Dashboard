# HR Analytics Power BI Dashboard

This dashboard analyzes employee attrition, workforce distribution, salary structure, and department-wise trends. It provides clear insights to help HR teams understand why employees leave and how to improve retention.

## Key Insights
- Attrition by department, gender, age, and job role
- Salary distribution across roles and experience levels
- Employee age group and education field trends
- Job satisfaction and work-life balance patterns
- KPIs: Total employees, active employees, attrition count & rate

## What the Dashboard Shows
- Attrition overview and trends
- Demographic analysis (age, gender, education)
- Salary and experience insights
- Department-wise performance metrics
- HR KPIs for decision-making
